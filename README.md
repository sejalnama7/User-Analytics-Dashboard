User Analytics Dashboard

A responsive analytics dashboard built using React, Vite, and Tailwind CSS. This project provides a clean and modern user interface for displaying business analytics and key performance metrics.

Features

- Responsive design for desktop and mobile
- Sidebar navigation
- Dashboard statistics cards
- Sales bar chart
- Device usage pie chart
- Recent orders table
- Search bar
- User profile section

Technologies Used

- React.js
- Vite
- Tailwind CSS
- Recharts
- React Icons

Installation

bash
npm install
npm run dev


 Project Structure

- Dashboard Layout
- Responsive Navbar
- Sidebar
- Statistics Cards
- Sales Chart
- Pie Chart
- Recent Orders Table
